# Project Summary
The LLM Gateway Platform is a multi-tenant SaaS solution that integrates advanced AI capabilities into various business applications. It efficiently routes requests to multiple AI model providers, offering a user-friendly chat interface, a visual editor for AI agents, and robust management tools for Model Configuration Protocol (MCP). The platform's scalable microservice architecture enhances operational efficiency and user engagement through sophisticated AI technologies. Recent updates have improved the user interface, particularly in workflow management and chat functionalities, and introduced a gaming feature that allows users to play classic retro games directly within the application.

# Project Module Description
- **Gateway**: Manages API routing and load balancing.
- **Auth Service**: Handles user authentication and session management.
- **Chat Service**: Processes chat requests and manages conversations with AI services, now featuring streaming output and a blinking cursor.
- **Agent Service**: Oversees the creation and execution of AI agents.
- **MCP Service**: Manages registration and operation of MCP tools.
- **Admin Service**: Administers API keys, model providers, and usage statistics.
- **Workflow Engine**: Provides a visual editor for workflow design.
- **Base Service**: A generic CRUD service to reduce code duplication across entity services.
- **Tenant Management**: Manages operations for multiple tenants, including creation and member administration.
- **Billing Service**: Monitors usage and manages billing based on token consumption.
- **Agent Execution Service**: Manages the lifecycle of agent execution, including scheduling and result tracking.
- **MCP Protocol Service**: Facilitates communication with MCP servers for tool discovery and invocation.
- **Model Experience Service**: Offers a dedicated interface for visualizing and interacting with AI models.
- **Game Service**: Embeds retro games, allowing users to play classic games directly within the platform, implemented using pure HTML5/JS/Canvas.
- **Landing Page**: A new entry point that automatically redirects users to the dashboard, enhancing user experience by eliminating authentication delays.

# Technology Stack
- **Frontend**: React 18, TypeScript, Vite, Tailwind CSS
- **Backend**: Python FastAPI, SQLAlchemy (async), Atoms Cloud
- **Database**: PostgreSQL
- **Cache**: Redis
- **Message Queue**: RabbitMQ
- **Code Editor**: Monaco Editor
- **Workflow Engine**: React Flow

# Usage
To set up the project, follow these steps:
1. **Install Dependencies**:
   - For the backend:
     ```
     cd app/backend
     pip install -r requirements.txt
     ```
   - For the frontend:
     ```
     cd app/frontend
     pnpm install
     ```
2. **Build the Project**:
   - For the backend:
     ```
     cd app/backend
     python -m py_compile services/chat_service.py routers/chat.py services/base_service.py
     ```
   - For the frontend:
     ```
     cd app/frontend
     pnpm run build
     ```
3. **Run the Application**:
   - Use Docker Compose to start the services:
     ```
     cd app/deploy
     docker-compose up
     ```
